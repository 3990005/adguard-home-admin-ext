AdGuard Home Admin Support

Get help with managing your AdGuard Home server

Getting Started

Requirements

  • AdGuard Home Server: You need a running AdGuard Home instance (self-hosted)
  • Network Access: Your iPhone must be able to reach your server (local network or remote)
  • Credentials: Your AdGuard Home admin username and password

Adding Your First Server

  1. Open AdGuard Home Admin
  2. Tap "Add Server"
  3. Enter a name for your server (e.g., "Home DNS")
  4. Enter the server URL (e.g., http://192.168.1.1:3000 or https://dns.yourdomain.com)
  5. Enter your AdGuard Home username and password
  6. Tap "Save"
Tip: We recommend using HTTPS for secure connections, especially when accessing your server remotely.

Frequently Asked Questions

I can't connect to my server

Check the following:

  • URL Format: Include the protocol and port, e.g., http://192.168.1.1:3000
  • Network: Ensure your iPhone is on the same network as your server (for local connections)
  • Firewall: Verify that the AdGuard Home port is accessible
  • Server Running: Confirm AdGuard Home is running on your server
  • Credentials: Double-check your username and password

I see "Failed to parse server response"

This usually means the app received an unexpected response. Common causes:

  • The URL points to something other than AdGuard Home
  • Your AdGuard Home version may be outdated—try updating to the latest version
  • A reverse proxy is modifying the response

How do I access my server remotely?

To access AdGuard Home outside your local network:

  • VPN: Connect to your home network via VPN (recommended)
  • Reverse Proxy: Set up a reverse proxy with HTTPS (e.g., nginx, Caddy)
  • Tailscale/ZeroTier: Use a mesh VPN service

We strongly recommend using HTTPS and authentication for any remote access.

What does the Live mode do in Query Log?

Live mode automatically refreshes the query log every 500ms, showing new DNS queries in real-time. This is useful for:

  • Troubleshooting connectivity issues
  • Seeing which domains a device is contacting
  • Verifying that blocking rules are working

Tap the "Live/Paused" button to toggle this feature.

How do I block services for a specific device?

  1. Go to Settings → Clients
  2. Tap on a configured client (or add a new one)
  3. Toggle off "Use Global Blocked Services"
  4. Select the specific services you want to block for that device
  5. Tap Save

Can I manage multiple AdGuard Home servers?

Yes! You can add multiple servers and switch between them:

  • Tap the server name in the top-left of the Dashboard
  • Select a different server or add a new one

Where are my credentials stored?

Your server credentials are stored securely in the iOS Keychain, which uses hardware-level encryption. They never leave your device except to authenticate with your own server.

The app shows outdated information

Pull down on any screen to refresh the data. You can also use the refresh button in the menu on the Dashboard.

Troubleshooting

Connection Issues Checklist:
  1. Verify AdGuard Home is running: visit the URL in Safari
  2. Check for typos in the server URL
  3. Try using the IP address instead of hostname
  4. Ensure you're including the correct port number
  5. Test on the same WiFi network as your server first
